Detailed Description

Class associated to a ViewingManager instance in order to parse and load XML configuration files for the setup.

Gets a reference to ViewingManager and calls the create_navigation, create_powerwall_user, create_ovr_user and create_desktop_user according to the settings read from the configuration file.
