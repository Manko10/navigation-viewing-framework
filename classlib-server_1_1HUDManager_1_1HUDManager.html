def lib-server.HUDManager.HUDManager.remove_from_coupling_display (   self,
  NAVIGATION 
)

Tells the HUDManager that a Navigation instance is now out of every coupling.

The HUDManager will then check for all users on the platforms and update their coupling status notifiers. Apart from that, a message is shown to all users in the group that NAVIGATION has decoupled from the group.

Parameters:
NAVIGATIONThe Navigation instance to be removed from all couplings.
